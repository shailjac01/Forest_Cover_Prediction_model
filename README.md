# Forest_Cover_Prediction_model

## Project Overview
This project predicts forest cover types using machine learning techniques based on environmental features such as elevation, slope, and distance to hydrology.

## Dataset
The dataset used is the **Covertype dataset**(downloaded from kaggle), which contains cartographic variables used to classify forest cover types.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Machine Learning Model
The model used for prediction is **Logistic Regression**.

## Model Evaluation
The model was evaluated using:
- Accuracy Score
- Confusion Matrix
- Classification Report

**Accuracy Achieved:** ~70%(using both oversampling and undersampling techniques)

## Project Workflow
1. Data loading
2. Data preprocessing
3. Feature scaling
4. Train-test split
5. Model training
6. Model evaluation
